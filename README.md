 Tech Stack & Features
**Layer	Technologies/Concepts Used**
Backend	                  ASP.NET Core 7 Web API, Entity Framework Core, LINQ, AutoMapper, Dependency Injection
Frontend	                ASP.NET Core MVC or Angular/React (You can choose), Razor Pages, Bootstrap
Database	                SQL Server, EF Migrations
Security	                JWT Authentication, Role-Based Authorization, Middleware
Hosting & DevOps	        GitHub, Azure App Service (optional), Docker (optional), Swagger for API testing
Extra Features	          Logging, Exception Handling, Email Notification, API Rate Limiting (optional)
